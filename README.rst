
===========
PHP Formula
===========

PHP is a widely-used general-purpose scripting language that is especially
suited for Web development and can be embedded into HTML.

Sample Pillars
==============

.. code-block:: yaml

    php:
      environment:
        enabled: true
        cache:
          engine: 'apc'
          shm_size: 128
          max_file_size: '10M'
